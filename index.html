Shrunked Image Resizer

for Thunderbird, Firefox and SeaMonkey

Shrunked is an extension that resizes your photos before you upload them to a website or email them. It greatly reduces the time spent waiting for photos to upload.

You can get Shrunked from addons.mozilla.org. If you got it from somewhere else, including my own website, it's not genuine. Visit a.m.o. and get a fresh copy.

Localization

You might be here because you'd like to see Shrunked in your language. Great!

To translate, you can:

  • Fork the Git repository, translate, and send me a pull request. Some knowledge of creating Mozilla add-ons is required.
  • Or, visit the project's page on Zoo, a website for add-on translation. (Please note that Zoo is under development, and could break at any time. It's mostly stable though.)

Contribute code

Want to add some functionality to Shrunked? Feel free to send me a pull request.

To get a working version of this repo in your Firefox profile, clone it into your extensions directory as shrunked@darktrojan.net and start Firefox.

cd [your profile dir]/extensions
clone git://github.com/darktrojan/shrunked.git shrunked@darktrojan.net

Contribute money

Add-ons are available for free, but developing and supporting them does take a considerable amount of time. Please consider making a donation to help me pay my bills. You can contribute any amount you like through PayPal via addons.mozilla.org. (Note though that for amounts smaller than about US$1, PayPal takes most of it.)
